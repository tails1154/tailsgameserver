# TailsGame Server



server for TailsGame services


## Running


`git clone https://github.com/tails1154/tailsgameserver`


`cd tailsgameserver`



`npm install`


and to run it,

`npm start`

or all at once


`git clone https://github.com/tails1154/tailsgameserver;cd tailsgameserver;npm install;npm start`
